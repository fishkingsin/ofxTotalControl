Installation
------------
Build Settings -> Other Linke Flags	-> add -lftd2xx

Dependencies
------------
i386:
http://www.ftdichip.com/Drivers/D2XX.htm
follow the instruction to install ftd2xx

arm:
spi

Compatibility
------------
p9813 i386,x86_64 only
elinux-tcl arm only
